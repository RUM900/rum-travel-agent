# 黄金路线模板库

每条路线均为实地验证过的行程，包含每日安排、预算参考、避坑提示。

## 川西小环线 · 6日经典自驾（成都进出）

**适合人群**：首次进川西、自驾爱好者、摄影爱好者
**最佳季节**：6-8月（草绿花海）、10月中-11月初（彩林季）
**不适合**：严重高原反应史者（最高海拔4200m）

### 每日安排

**D1 成都→康定→新都桥（车程6h）**
- 上午：成都出发，雅康高速3.5h到康定
- 中午：康定午餐，推荐"玛吉阿米藏餐厅"（人均70元，牦牛肉包子必点）
- 下午：翻越折多山垭口（4298m，第一个高反检测点，停留不超30分钟）
- 傍晚：抵达新都桥，拍摄贡嘎雪山日落（最佳机位：新都桥加油站后山）
- 住宿：新都桥云漫山庄 380-580元/晚（有氧气房，海拔3300m）

**D2 新都桥→塔公→墨石公园→丹巴（车程4h）**
- 上午：塔公草原（免费），塔公寺（20元），雅拉雪山背景拍摄点
- 中午：塔公镇午餐，"雪域藏餐"（人均50元，酥油茶正宗）
- 下午：墨石公园（门票60元+观光车20元），游览2-3h
- 傍晚：抵达丹巴县，住甲居藏寨内民宿
- 住宿：甲居藏寨·云端藏家 200-350元/晚（含早餐，藏式老房子改造）

**D3 丹巴→四姑娘山双桥沟（车程3h）**
- 上午：甲居藏寨晨拍（日出时分光影最佳，门票50元）
- 中午：丹巴县城午餐
- 下午：抵达四姑娘山双桥沟，游览4-5h（门票+观光车150元）
- 核心站点：红杉林（海拔3840m）→布达拉峰→四姑娜措→人参果坪
- 住宿：四姑娘山镇 雲上·山居 400-600元/晚（海拔3200m）

**D4 四姑娘山长坪沟全天**
- 全天：长坪沟徒步（门票+观光车90元），栈道往返14km约6-7h
- 体力有限者走到枯树滩（3km处）即可返回
- 关键提示：沟内无补给站，自备午餐和水；14:00后不建议深入
- 住宿：同D3

**D5 四姑娘山→卧龙→映秀→成都（车程4h）**
- 上午：途经猫鼻梁观景台（拍摄四姑娘山四峰全景，免费）
- 中午：卧龙镇午餐
- 下午：可选卧龙熊猫苑（门票90元，比成都熊猫基地人少80%）
- 傍晚：返回成都

**D6 成都休整/市区**
- 锦里+武侯祠（通票60元）或宽窄巷子（免费）+人民公园喝茶
- 晚餐推荐："小龙翻大江"火锅（人均100-150元，提前2h排号）

### 预算参考

| 项目 | 人均费用 |
|------|---------|
| 租车（SUV，5天） | 1500-2500元 |
| 油费+过路费 | 800-1000元 |
| 住宿（5晚） | 1500-2500元 |
| 门票 | 400-500元 |
| 餐饮 | 1000-1500元 |
| **总计** | **5200-8000元** |

### 避坑提示
- 折多山每年11月-次年3月可能大雪封山，出行前查甘孜交警公众号
- 新都桥海拔3300m，第一晚不要洗澡洗头（加速高反）
- 墨石公园正午强光下拍照效果差，15:00后最佳
- 丹巴到四姑娘山走G350中国熊猫大道，部分路段雨季有落石
- 全程最高点折多山4298m，在康定买氧气瓶（20-30元/罐，景区卖60+）

## 云南经典 · 8日昆大丽泸（飞机进出）

**适合人群**：蜜月旅行、闺蜜出游、首次来云南
**最佳季节**：3-5月（春暖花开）、10-11月（秋高气爽避开雨季）
**不适合**：对商业化古城零容忍者

### 每日安排

**D1 抵达昆明→高铁到大理（2h）**
- 昆明长水机场→昆明南站（地铁1h或打车40分钟）
- 高铁昆明南→大理站（2h，二等座145元，每小时一班）
- 住宿：大理古城内 既下山SUNYATA 600-900元/晚
- 晚餐：古城"醉琉璃"（人均80元，酸辣鱼+水性杨花汤）

**D2 环洱海自驾全天**
- 租车：Smart/甲壳虫 200-300元/天，电瓶车80元/天（只能跑半圈）
- 顺时针环湖：古城→喜洲古镇→双廊→挖色→小普陀→古城
- 喜洲古镇（免费）：白族民居+转角楼拍照+喜洲粑粑（10元/个）
- 午餐：喜洲"翰林餐厅"（人均60元，生皮慎点）
- 双廊：杨丽萍太阳宫（预约制，公众号提前3天约，免费参观限30分钟）
- 挖色码头：日落前1h到达，拍摄洱海最美日落
- 全程120km，走走停停需8-10h

**D3 大理→沙溪古镇（包车1.5h）**
- 沙溪古镇（免费）：茶马古道上唯一幸存的古集市
- 寺登街、古戏台、玉津桥（日出时拍摄最佳）
- 午餐：沙溪"古道惠风"（人均50元，火腿炒菌子）
- 下午：黑惠江畔骑马（100元/人/小时）
- 住宿：沙溪·既下山 500-700元/晚

**D4 沙溪→丽江（大巴2.5h，50元）**
- 抵达丽江大研古城
- 注意：古城维护费50元（8:30-19:00查票），但17:00后通常不查
- 住宿推荐束河古镇而非大研古城（安静+性价比高）：悦榕庄旁·束河花间堂 400-600元/晚
- 晚餐：束河"小锅巴纳西美食"（人均70元，纳西烤鱼+鸡豆凉粉）

**D5 玉龙雪山一日**
- 关键！大索道票（冰川公园，4680m）需提前3天在"丽江旅游集团"小程序抢，每天7:00放票
- 大索道+观光车 140元，进山费100元
- 抢不到大索道→备选云杉坪索道（60元，3240m）+蓝月谷
- 装备：氧气瓶（丽江买20-30元，索道站卖60+）、厚外套（夏季也需）
- 住宿：同D4

**D6 丽江→泸沽湖（包车3.5h，拼车80元/人）**
- 途经丽宁十八弯（晕车者提前吃晕车药！这段弯道极多）
- 泸沽湖门票70元
- 住宿：里格半岛 泸沽湖·半岛阳光 500-800元/晚（湖景房）
- 晚餐：里格"阿妈土鸡火锅"（人均80元）

**D7 泸沽湖环湖+猪槽船**
- 环湖方式：包车300元/天或租电动车100元（约70km，注意电量）
- 猪槽船游湖：里格码头→王妃岛→亲爱的客栈（100元/人，约2h）
- 走婚桥+草海（免费），夏季草海绿、秋季金黄
- 格姆女神山索道70元（俯瞰泸沽湖全景）
- 住宿：同D6

**D8 泸沽湖→丽江→返程**
- 泸沽湖→丽江机场（包车4h）或丽江三义机场→返程
- 建议订15:00以后的航班

### 预算参考

| 项目 | 人均费用 |
|------|---------|
| 大交通（往返机票） | 1500-3000元 |
| 云南省内交通 | 800-1200元 |
| 住宿（7晚） | 2800-4500元 |
| 门票+索道 | 600-800元 |
| 餐饮 | 1500-2000元 |
| **总计** | **7200-11500元** |

### 避坑提示
- 大理古城内的"白族三道茶"表演大多收费且质量参差，不如去喜洲看真实的白族民居
- 丽江古城内超过200元/晚的客栈未必比束河300元的好，大研古城溢价严重
- 玉龙雪山大索道票夏季和节假日极难抢，做行程时预留云杉坪备选方案
- 泸沽湖盘山路晕车概率极高，务必提前备药

## 西北经典 · 5日青甘小环线（西宁进出）

**适合人群**：首次去西北、假期5天内、喜欢自然风光
**最佳季节**：6-8月（青海湖油菜花+气温舒适）、9-10月（秋高气爽但早晚冷）
**不适合**：膝盖/腰椎伤病者（部分路段颠簸）

### 每日安排

**D1 西宁→塔尔寺→青海湖（车程3h）**
- 上午：塔尔寺（门票70元），藏传佛教格鲁派六大寺院之一，游览2-3h
- 中午：湟中县午餐
- 下午：沿G109西倒一级路前往青海湖，途经日月山（免费观景台）
- 傍晚：青海湖二郎剑景区附近看日落
- 住宿：青海湖·九龙客栈 350-500元/晚（离湖边步行5分钟）
- 关键：青海湖每年7月油菜花期最盛，8月开始凋谢

**D2 青海湖→茶卡盐湖→德令哈（车程5h）**
- 上午：青海湖日出（夏季约6:15），走环湖东路（车少景美，免费）
- 中午：黑马河镇午餐（青海湖边小镇，餐饮贵且一般，吃饱就行）
- 下午：茶卡盐湖（门票60元，小火车单程50元），拍照需晴天+before 16:00
- 穿鲜艳长裙拍照效果好，景区租鞋套10元（赤脚踩盐湖硌脚）
- 傍晚：驱车2h到德令哈住宿
- 住宿：德令哈·海西宾馆 200-300元/晚

**D3 德令哈→大柴旦翡翠湖→敦煌（车程6h）**
- 上午：大柴旦翡翠湖（目前免费，即将收费），比茶卡人少+颜色更丰富
- 中午：大柴旦镇午餐
- 下午：驱车穿越当金山口（海拔3648m）进入甘肃
- 傍晚：抵达敦煌
- 住宿：敦煌市区 敦煌山庄 400-600元/晚（摘星阁露台可看鸣沙山）

**D4 莫高窟+鸣沙山月牙泉**
- 上午：莫高窟（A类票238元，需提前30天在小程序预约！含8个洞窟+两部电影）
- B类票100元（含4个洞窟，提前1天约即可，但旺季也很难抢）
- A类票优先预约上午场次（光线好+人少），全程约3h
- 下午：16:00后去鸣沙山月牙泉（门票110元），中午沙子烫到无法行走
- 骑骆驼100元/人（约40分钟），滑沙20元/次
- 日落时间约20:00，在沙山顶看日落是敦煌必做清单TOP1
- 住宿：同D3

**D5 敦煌→返程**
- 上午：敦煌博物馆（免费，需预约），了解丝绸之路历史（1.5h）
- 或选择西线：玉门关（40元）+雅丹魔鬼城（50元+70元观光车），车程单程2h
- 下午：返程航班

### 预算参考

| 项目 | 人均费用 |
|------|---------|
| 包车/拼车（4天） | 1200-2000元 |
| 住宿（4晚） | 1300-2100元 |
| 门票 | 500-700元 |
| 餐饮 | 800-1200元 |
| **总计（不含往返大交通）** | **3800-6000元** |

### 避坑提示
- 茶卡盐湖旺季（7-8月）人多到怀疑人生，优先选大柴旦翡翠湖替代
- 莫高窟A类票暑假和国庆极其难抢，务必提前30天卡点预约
- 青海湖不要进"牧民私设的湖边入口"（容易被宰），走正规景区或环湖东路
- 大柴旦到敦煌途经无人区，出发前加满油（中间加油站少且可能没油）
- 西北干燥+紫外线极强，防晒霜+润唇膏+墨镜是保命三件套

## 华东慢游 · 4日苏州杭州双城（高铁出行）

**适合人群**：带长辈出行、文化爱好者、短假期
**最佳季节**：3-4月（江南春色+龙井新茶）、10-11月（桂花香+天气稳定）

### 每日安排

**D1 抵达苏州→拙政园→平江路**
- 拙政园（门票80元），建议请讲解（100元/小时）或租讲解器（20元），否则只能看热闹
- 游览1.5-2h，最佳入园时间：7:30开门即入（8:30后旅行团涌入）
- 中午：苏州博物馆（免费，需提前7天在公众号预约），贝聿铭封山之作
- 下午：平江路（免费），沿河老街区，评弹茶馆推荐"伏羲茶馆"（88元/人含茶）
- 住宿：平江路附近 苏州文旅花间堂·探花府 500-800元/晚

**D2 苏州→杭州（高铁1.5h，二等座120元）**
- 上午：虎丘（门票70元），"到苏州不游虎丘乃憾事也"，游览1.5h
- 中午：山塘街新民桥菜市场（比观前街更地道），推荐"荣登楼"生煎（20元/8只）
- 下午：高铁到杭州
- 住宿：西湖周边 杭州法云安缦附近·灵隐民宿 500-800元/晚

**D3 西湖环湖+灵隐寺**
- 上午：灵隐寺（飞来峰45元+灵隐寺香花券30元），7:30开门即入避开人潮
- 中午：灵隐寺十方苑素面（25元/碗，11:00前到不然要排队）
- 下午：西湖环湖骑行（共享单车即可），推荐逆时针：断桥→白堤→苏堤→雷峰塔
- 雷峰塔（门票40元），上塔俯瞰西湖全景
- 傍晚：长桥公园拍雷峰夕照（西湖十景之一，日落前30分钟到达）

**D4 龙井+返程**
- 上午：龙井村+九溪十八涧徒步（免费），清明前后可看采茶
- 龙井村买茶避坑：真正的狮峰龙井价格至少500元/斤起步，路边"现炒现卖"大多是外地茶
- 中午：龙井村"茶人村"（人均120元，龙井虾仁+茶香鸡）
- 下午：返程

### 预算参考

| 项目 | 人均费用 |
|------|---------|
| 苏州-杭州高铁 | 120元 |
| 住宿（3晚） | 1500-2400元 |
| 门票 | 350-400元 |
| 餐饮 | 800-1200元 |
| **总计（不含大交通）** | **2800-4100元** |

### 避坑提示
- 拙政园门口拉客的"低价一日游"全是要带去购物点的，直接无视
- 西湖边的"楼外楼"名大于实，不如去"杭州酒家"（人均80元，本地人更认可）
- 苏州公交卡不支持支付宝直接刷，需下载"苏e行"APP
